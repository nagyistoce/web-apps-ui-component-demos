= Radio Buttons =

Demonstrates the Radio Button component.

== Design ==

The Buttons - radio page demonstrates the usage of the radio button inputs and their characteristics. The example displays radio buttons used in lists, inactive radio buttons, grouped radio buttons and default radio buttons.

For more information on Radio Buttons, please see the 
[http://www.developer.nokia.com/Resources/Library/Series_40_web_apps_library/#!design-library-intoduction/checkboxes-and-radio-buttons.html Series 40 web apps design library].

==== Radio Buttons help text: ====
Buttons – radio demonstrate radio button groups. “Normal” tab shows a list with a single radio button group, “Inactive” tab displays an inactive radio button group, and “Grouped” tab shows two independent radio button groups, separated by a header. A radio button group requires always one item to be selected.

[[Image(radio_buttons.png)]]

== Implementation ==

The example code snippet shows creating two radio button list items (with one being selected by default) and belonging to the same mutually exclusive group.

Please note, that the horizontal aligning of the elements is usually done with tables in S40 Web Apps, as the CSS "float" style formatting isn't supported.

{{{
<div class="checkable_list_item"> 										
	<table class="fillWidth" cellpadding="0" cellspacing="0">
		<tr>
			<td class="checkable_item_cell_left">Item 1</td>
			<td class="checkable_item_cell_right">
				<input type="radio" name="tab_group_1" checked="checked" />
			</td>
		</tr>
	</table>
</div>
<div class="checkable_list_item">
	<table class="fillWidth" cellpadding="0" cellspacing="0">
		<tr>
			<td class="checkable_item_cell_left">Item 2</td>
			<td class="checkable_item_cell_right">
				<input type="radio" name="tab_group_1" />
			</td>
		</tr>
	</table>
</div>
}}}

The CSS that defines, how the radio button list items look like:

{{{
.checkable_list_item {
	height: 40px;
	background-color: #fff;
	padding: 0px 8px 0px 8px;
	font-weight: bold;
	border-bottom: 1px #e7e7e7 solid;
	font-size: 16px;
}

.checkable_item_cell_left, .category_left  {
	height: 40px;
	text-align: left;
}

.checkable_item_cell_right, .category_right{
	height: 40px;
	text-align: right;
}
}}}
