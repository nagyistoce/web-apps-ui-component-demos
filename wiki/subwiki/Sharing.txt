= Sharing =

The Sharing example demonstrates the platform's handling of an sms: URI scheme and sharing a status update through Facebook.

== Design ==

The Sharing example is simply a page with pre-defined status update text and a two buttons for sharing the update either with a text message or through Facebook.

For more information on sharing content through social networks, please see the 
[http://www.developer.nokia.com/Resources/Library/Series_40_web_apps_library/#!design-library-intoduction/sharing-content.html Series 40 web apps design library].

==== Sharing help text: ====
Sharing content illustrates that sharing links either opens a related web link (Facebook in the example), or an internal API for sending the content via SMS.

[[Image(sharing.png)]]

== Implementation ==

The Sharing mini web app is implemented just with few simple anchor links. Check the code below.

Please note, that the horizontal aligning of the elements is usually done with tables in S40 Web Apps, as the CSS "float" style formatting isn't supported.

{{{
<!--SharingContent mini app-->
<div id="sharingContent" class="slider_section hide">
  <div class="list_section_header">Sharing</div>
  
  <div class="text_block share_text_block">
    Lorem ipsum dolor sit amet, consectetur adipisicing elit.
  
  </div>
  <table class="share_table" cellspacing="5" cellpadding="0">
    <tr>
      <td id="share_title">Share</td>
      <td class="alignR">
        <a href="sms:?body=LoremIpsumDolorSitAmet">
          <img src="img/share_sms.png" />
        </a>
      </td>
      <td class="alignR">
        <a href="http://www.facebook.com/sharer.php?u=LoremIpsumDolorSitAmet">        
          <img src="img/share_facebook.png" />
        </a>
      </td>
    </tr>
  </table>
  
</div>
<!--End SharingContent mini app-->
}}}
