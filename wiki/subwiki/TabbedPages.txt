= Tabbed Pages =

The Tabbed Pages example demonstrates how to implement a real application using tabs.

== Design ==

The Tabbed Pages example is designed to represent a life like application with three different pages, each on its own tab. The pages are filled with static list content.

The S40 Web Apps design guide recommends maximum of 4 tabs to be shown simultaneously.

For more information on Tabs, please see the 
[http://www.developer.nokia.com/Resources/Library/Series_40_web_apps_library/#!design-library-intoduction/tabs.html Series 40 web apps design library].

==== Tabbed Pages help text: ====
Tabbed pages illustrate the basic (horizontal) navigation for web apps. It means that all views are on the same hierarchical level. Back will return to the main view.

[[Image(tabbed_pages.png)]]

== Implementation ==

The Tabbed Pages example app code has been divided into four pieces. The most relevant pieces are the two first, .

Please note, that the horizontal aligning of the elements is usually done with tables in S40 Web Apps, as the CSS "float" style formatting isn't supported.

==== Main level division and Tab bar ====

The main level structure defines three tabs with images. Each tab is clickable and will open up the related content to that tab. Opening the tab's page is done with the MWL setGroupTarget method (by showing the tabbedPagesTab_X_content and hiding others).
{{{
<!--TabbedPages mini app-->
<div id="tabbedPages" class="slider_section hide">
  <table cellpadding="0" cellspacing="0" class="fill" summary="Tab Table">
    <tr id="tabbedPagesTabBar">
      <td id="tabbedPagesTab_1" class="tab tab_3_tabs tabbed_pages_tab_selected"
        onclick="
          mwl.setGroupTarget('#tabbedPagesTabControlContent', '#tabbedPagesTab_1_content', 'show', 'hide');
          mwl.setGroupTarget('#tabbedPagesTabBar', '#tabbedPagesTab_1', 'tabbed_pages_tab_selected', 'tab_not_selected');
        ">
        <img src="img/tab_list.png" alt="smile" />
      </td>
      <td id="tabbedPagesTab_2" class="tab tab_3_tabs tab_not_selected"
        onclick="
          mwl.setGroupTarget('#tabbedPagesTabControlContent', '#tabbedPagesTab_2_content', 'show', 'hide');
          mwl.setGroupTarget('#tabbedPagesTabBar', '#tabbedPagesTab_2', 'tabbed_pages_tab_selected', 'tab_not_selected');
        ">
        <img src="img/tab_group.png" alt="smile" />
      </td>
      <td id="tabbedPagesTab_3" class="tab tab_3_tabs_right tab_not_selected"
        onclick="
          mwl.setGroupTarget('#tabbedPagesTabControlContent', '#tabbedPagesTab_3_content', 'show', 'hide');
          mwl.setGroupTarget('#tabbedPagesTabBar', '#tabbedPagesTab_3', 'tabbed_pages_tab_selected', 'tab_not_selected');
        ">
        <img src="img/tab_contact.png" alt="smile" />
      </td>
    </tr>
  </table>
}}}
