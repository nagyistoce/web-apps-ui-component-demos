= Buttons =

Demonstrates the basic Button component.

== Design ==

The Buttons page demonstrates different ways of defining button sizes and placement.

For more information on Buttons, please see the 
[https://www.developer.nokia.com/Resources/Library/Series_40_web_apps_library/#!design-library-intoduction/buttons.html Series 40 web apps design library].

==== Buttons help text: ====
Buttons show a variety of buttons grouped in a row. The group can have a headline and at maximum three buttons with different spacing in a row.

[[Image(buttons.png)]]

== Implementation ==

The example code snippet creates two buttons in a row. Please note, that the horizontal aligning of the elements is usually done with tables in S40 Web Apps, as the CSS "float" style formatting isn't supported.

{{{
<table class="fillWidth" cellpadding="0" cellspacing="0">
	<tr>
		<td class="alignL"><input type="button" name="button1"
			class="button button_2_in_row_width" value="BUTTON1"></input>
		</td>
		<td class="alignR"><input type="button" name="button2"
			class="button button_2_in_row_width" value="BUTTON2"></input>
		</td>
	</tr>
</table>
}}}

The CSS part merely defines the buttons' widths and text-aligns:

{{{
.button_2_in_row_width {
	width: 100px;	
}

.alignR {
	text-align: right
}

.alignL {
	text-align: left
}
}}}
